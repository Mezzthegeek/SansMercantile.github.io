About the Role

You’ll architect and build the digital backbone of Sans Mercantile—from our AI-driven advisory dashboard to our high-frequency trading interface. This is your chance to shape a platform that redefines financial access and intelligence. You will be responsible for creating scalable, responsive web applications, integrating APIs for trading, analytics, and user management, and optimizing for performance, security, and exceptional user experience.

Key Responsibilities

  • Design, develop, and maintain responsive and secure web applications using modern full-stack technologies.
  • Implement and integrate APIs for seamless data flow between trading systems, analytics engines, and user dashboards.
  • Ensure high performance, scalability, and uptime of the platform through robust architecture and CI/CD pipelines.
  • Collaborate with AI/ML engineers and quantitative analysts to translate complex financial models into user-friendly interfaces.
  • Contribute to the entire software development lifecycle, from conceptualization to deployment and maintenance.
  • Uphold strong cybersecurity practices and compliance standards in all development efforts.

Required Qualifications

  • Proficient in full-stack JavaScript development (e.g., React.js, Node.js) with experience in backend frameworks.
  • Experience with database technologies (e.g., MongoDB, PostgreSQL) and cloud deployment (e.g., AWS, GCP).
  • Strong understanding of RESTful APIs and secure integration practices.
  • Familiarity with fintech platforms, trading dashboards, or financial data systems is a significant plus.
  • Startup mindset: autonomous, proactive, fast-paced, and quality-driven.
  • Excellent problem-solving skills and attention to detail.
